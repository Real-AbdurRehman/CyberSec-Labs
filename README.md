The repository is organized into distinct categories to separate theory from hands-on lab environments:

*   **`📁 tryhackme/`** – Detailed write-ups, room completions, and methodology notes from TryHackMe learning paths.
*   **`📁 institutional-coursework/`** – Structured notes, labs, and assignments from my formal cybersecurity education.
